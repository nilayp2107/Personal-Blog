Machine Learning

The term Machine Learning was first used by Arthur Samuel in the year 1959. The first formal definition was given by Tom M. Mitchell.

The purpose of AI is to make computers learn from experience (no human intervention) and understand the world in terms of hierarchy of concepts. Thus the term AI Deep Learning.

Classic Machine Learning

Model and Cost Function

Basically, machine learning is allowing computers to learn to do tasks without being programmed explicitly to do so.
We will look at the mathematical representation of supervised learning problems by building the hypothesis function and cost function to caluculate the error.
In supervised learning we need to build a relationship hypothesis between X and Y.

x[i] = Set of input variables 1 ≤ i ≤ m
y[i] = Set of output variable 1 ≤ i ≤ m

Training Set

The set x[i] and y[i] together are called training set and a single pair is called training pair.
X is {x[1], x[2],..., x[m]}
Y is {y[1], y[2],..., y[m]}

Cost Function

It calculates the average difference of all the results of hypothesis with the input from X and the actual outputs from Y.
J(θ1,θ2) = 1/2m ∑mi=1 (ŷi - yi)2
