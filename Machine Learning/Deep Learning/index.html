Deep Learning

History

IBM's DeepBlue machine defeated world chess champion Garry Kasparov in 1997. However, the chess world is rather small.

Knowledge was first hard coded using formal language with logical inference rules. This was known as knowledge based approach.

Then Machine Learning was developed to have the capacity to extract patterns from raw data, automating the need to actually hard code the knowledge base. E.g. We could use Logistic Regression to determine whether we should use cesarean delivery or not, naive Bayes to classify mail as spam or not, etc.

With this the Representation of data also hold a significance. The data which is used to train the model using Machine learning alogirthms have attributes called features

Finding proper representation of data is sometimes time consuming, so people came with the idea of using ML to find good set of features. This is known as Representation Learning.
E.g. Autoencoders which consists of encoders and decoders are used to find good representation of data.

Factors of Variation are the features which explains the data and influences the output.

Steps Involved in Building Deep Learning Models

Goal: Design Models that generalise well to new, previously unseen data
Step 1: Split the data into a "training set" and a "test set"
Step 2: Train the model on "training set" to try to minimize prediction error on it.
Step 3: Apply the trained model on "test set" to measure the generalization error.

Evaluating Classification Models: Confusion Matrix

A confusion matrix or a contigency table or an error matrix consists of rows and columns with the predicted and actual numbers as shown.

Actual Positive Actual Negative
Predicted Positive TP FP
Predicted Negative FN TN

Where TP = True Positive, FP = False Positive, FN = False Negative, TN = True Negative

Accuracy = TP+TN⁄TP+FP+FN+TN

Precision = TP⁄TP+FP

Recall = TP⁄TP+FN

F1 Score = 2*Precision*Recall⁄Precision+Recall

For email detection it's better to use Precision whereas in cancer detection it's better to use Recall.

Biological neurons: Inspiration

The main inspiration came from the animal's computing machinery.
Neuron: The basic unit in the nervous system for receiving, processing and transmitting information. e.g. messages such as "hot", "loud", "spicy".
A nematode has 302 neurons, whereas a human has around 100 billion neurons.

  • When the input signals exceed a certain threshold within a short period of time, a neuron "fires"
  • Neuron "firing" is an "all-or-none" process where either a signal is sent or nothing happens.
