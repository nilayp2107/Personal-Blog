Python

Python is a high-level programming language developed by Guido Van Rossum. The name was kept after the English TV series "Monty Python's Flying Circus. It's is an interpreted language.
There are two main modes of Python Programming:

  1. Scipt Mode Programming (Notebook)
  2. Interactive Mode Programming (.py file)
