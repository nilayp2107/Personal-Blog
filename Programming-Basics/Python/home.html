Python

Python is a high-level programming language developed by Guido Van Rossum. The name was kept after the English TV series "Monty Python's Flying Circus. It's is an interpreted language.
There are two main modes of Python Programming:

  1. Script Mode Programming (Notebook)
  2. Interactive Mode Programming (.py file)

Identifiers

Names given to identify variables, fuctions, class, module or any othe object.
You can't use the reserved keywords as identifiers.

Indentations

All the statements within the block should be indented the same amount.
Multiple statements can be written in a single line using semicolon (;).

Data Types

Python supports five data types:
  1. Numbers
  2. Strings
  3. Lists
  4. Tuples
  5. Dictionaries
Numbers:

Python supports three kinds of number data types:

  1. Integers (Signed)
  2. Float
  3. Complex

Strings:

Contiguous set of characters represented in the quotation marks. We can use slice operator with indexes starting at 0 in the beginning of the string and working their way from -1 to the end. + sign is the concatenation operator and * is the repetition operator.

Lists:

Lists contain items seperated by ',' and enclosed within square brackets []. Items in a list can be of different data types, unlike C++.

Tuples:

Tuples are similar to lists but they are enclosed within paranthesis (). The elements and size of the tuple can't be changed.

Dictionaries:

Dictionaries are enclosed within curly braces {}. It consists of key-value pairs. The key is unique and the value can be accessed by the key.

Operators

In the statement "4+5", the '+' operator manipulates the value of the operands which are 4 and 5 in this case. Python supports different types of Operators.

Arithmetic Operators:

The basic arithmetic operators are addition, subtraction, division, multiplication, modulus, exponent and floor division(//).

Comparison Operators:

These types of operators compare the values on both the sides and decide the relation between them. They are called relational operators.

Assignment Operators:

Assignment operator assigns the value on the right hand side to the variables on the left hand side.
The different assignment operators are equal to, add and, subtract and, multiple and, divide and, modulus and, and exponential and.

Logical and Membership Operators:

The Logical operators supported in python are logical AND, logical OR, and logical NOT.
The Membership operators are in and not in operators.

Python Decision Making

The basic decision making statements in python are as follows:

  • if statement
  • If...else statement
  • If...elif...else statement
  • Nested if statement
