NumPy

Numerical Python or NumPy is a library containing array objects and routines to process those arrays. NumPy is generally used with SciPy and MatlabPlots for generating mathematical functions and plotting.
The most important object defined in NumPy is an N-dimensional array type called ndarray.

Array Attributes of NumPy

  • ndarray.shape
  • ndarray.reshape
  • ndarray.arange
  • numPy.itemsize
  • np.zeros
  • np.ones
  • np.empty(shape, dtype, order)
  • numpy.linspace(start, stop, num, endpoint, retstep, dtype)

NumPy MatplotLib

Matplotlib is a plotting library for Python. It is an opensource alternative for MATLAB.
We include the library in the following way: 

from matplotlib import pyplot as plt

For using the following syntax are used: 

        plt.title(Title)
        plt.xlabel(x)
        plt.ylabel(y)
        plt.plot(x,y)
        plt.show()

Subplot
