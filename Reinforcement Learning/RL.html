Types of Machine Learning

  • Supervised Learning

    • We have to find a function 'f' using x and y provided.

    Dataset: Static Dataset

    Goal: Classification, Regression

  • Unsupervised Learning

    • We have to find a function 'f' only using the x provided.

    Dataset: Static Dataset

    Goal: Clustering

  • Reinforcement Learning

    • This is a totally different case. We are required to find 'f' using x and z provided. The 'f' will later be used to find y.

    Dataset: Data from Dynamic Environment

    Goal: Best sequence of actions for optimal outcome

What is Reinforcement Learning?

It is a part of Artificial Intelligence. In Reinforcement Learning, an agent is put in an environment and it learns to behave in that environment by performing certain actions and observing the rewards which it gets from those actions.

Process of Reinforcement Learning

  1. The agent is able to observe the current state of the environment.
  2. From that observed state, it decides which actions to take.
  3. The Environment changes state and produces a reward for that action.
  4. The agent receives both the state and the reward.
  5. Using the information the agent determines whether the action was good and should be repeated, or if it was bad and should be avoided.
    6. This obserbation-action-reward cycle continues until learning is complete.

Agent

It is a piece of software that explores, interacts with and learn from the environment
